
### 步骤2: 登录系统
```
访问: http://localhost:3004/
点击右上角"登录"按钮
输入"张建波"（总监权限）
```

### 步骤3: 查看项目周报
```
自动显示4个项目卡片
查看项目状态（红黄绿灯）
查看项目阶段
查看核心产出、行动、风险
```

### 步骤4: 拖拽排序
```
鼠标悬停在项目卡片上
看到右上角的拖拽手柄（⋮⋮）
拖拽项目卡片调整顺序
排序自动保存
```

### 步骤5: 编辑周报
```
点击项目卡片底部的"编辑周报"按钮
打开周报编辑对话框
修改状态、阶段
添加/删除任务项（风险、行动、决策、协作）
填写各项内容
点击"保存"
```

### 步骤6: 查看可视化看板
```
点击"可视化看板"标签
查看概览卡片（4个统计指标）
查看项目状态分布图
查看项目阶段分布图
查看风险项目列表
```

### 步骤7: 周次切换
```
使用顶部的周导航（◀ ▶）
切换到不同周次
查看不同周次的数据
点击"本周"快速返回当前周
```

### 步骤8: 其他功能
```
创建新项目 - 点击"录入数据"
查看历史记录 - 点击"历史记录"

```
