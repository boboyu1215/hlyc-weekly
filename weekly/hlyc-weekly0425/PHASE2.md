## 🎯 功能演示流程

### 完整使用流程：

1. **生成测试数据**
   - 访问 http://localhost:3004/test-data.html
   - 点击"生成测试数据"按钮
   - 自动创建4个项目和周报数据

2. **登录系统**
   - 访问 http://localhost:3004/
   - 点击右上角"登录"按钮
   - 输入"张建波"（总监权限）

3. **查看项目周报**
   - 自动显示4个项目卡片
   - 查看项目状态（红黄绿灯）
   - 查看项目阶段
   - 查看核心产出、行动、风险

4. **编辑周报**
   - 点击项目卡片底部的"编辑周报"按钮
   - 打开周报编辑对话框
   - 修改状态、阶段
   - 添加/删除任务项
   - 填写各项内容
   - 点击"保存"

5. **创建新项目**
   - 点击"录入数据"标签
   - 填写项目信息
   - 保存

6. **查看历史记录**
   - 点击"历史记录"标签
   - 查看所有操作的时间线

7. **切换周次**
   - 使用顶部的周导航（◀ ▶）
   - 切换到不同周次查看周报

8. **主题切换**
   - 点击右上角的主题按钮（🌙/☀️）
   - 在亮色和暗色主题之间切换

## 📊 数据结构

### 周报快照结构
```typescript
interface WeeklySnapshot {
  status: 'r' | 'y' | 'g';           // 状态
  stage: number;                      // 阶段（0-12）
  risk: TaskItem[];                   // 风险列表
  coreOutput: string;                 // 核心产出
  coreAction: TaskItem[];             // 核心行动
  decision: TaskItem[];               // 决策事项
  crossDept: TaskItem[];              // 跨部门协作
  next?: string;                      // 下周计划
  incident?: string;                  // 突发事件
  knowledge?: string;                 // 知识沉淀
  _ts?: number;                       // 时间戳
  _fieldTs?: Record<string, number>;  // 字段时间戳
  _savedWk?: string;                  // 保存的周次
}

interface TaskItem {
  text: string;      // 任务描述
  dueDate: string;   // 截止日期
}
```
